### const 用于定义常量，定义之后不能修改，不能再次赋值，在程序运行期间不会改变


*  定义一个常量

* 批量声明常量 
```
const (
    a = 1
    b = 2
    c = 3
)
```
* 批量声明常量时，如果某一行没有写=，那么就和上一行一致

```
const (
    a = 1
    b
    c
)
```
#### iota
* 在const关键字出现时将被重置为0；
* const中每增加一行常量声明，将使 iota 计数一次
* 类似枚举
```
const (
   b1 = iota //0
   b2        //1
   _         //2
   b3        //3
)
```

```
const (
   c1 = iota //0
   c2 = 100  //100
   c3        //100
   c4        //100
)
```

```
const (
   d1 = iota //0
   d2 = 100  //100
   d3 = iota //2
   d4        //3
)

```

#### 多个常量声明在一行

```
const (
   d1, d2 = iota + 1, iota + 2 //1 2
   d3, d4 = iota + 1, iota + 2 //2 3
)
```
